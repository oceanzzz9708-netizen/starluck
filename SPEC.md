# 星座运势查询 - 规范文档

## 1. 项目概述

- **项目名称**: 感情线+事业线 每周运势查询
- **项目类型**: 前端H5 + Flask后端
- **核心功能**: 用户选择"感情线"或"事业线"，填写星座信息，实时调用LLM生成个性化运势分析
- **目标用户**: 关注星座、想要了解感情和职场运势的年轻人

## 2. 设计风格

### 视觉风格
- **色调**: 深邃星空渐变背景（深蓝/紫色）+ 金色/粉色点缀
- **字体**: 优雅现代的中文字体
- **风格**: 神秘感、星空感、科技感

### 布局
- 全屏单页应用
- 顶部Tab切换「感情线」和「事业线」
- 输入区收集用户信息
- 结果区展示运势分析卡片

## 3. 功能模块

### 3.1 感情线运势

#### 输入字段
- 【我的星座】{my_sign} - 必选，12星座选择器
- 【对方星座】{crush_sign} - 必选，12星座选择器
- 【当前阶段】{situation} - 必选，暗恋/暧昧/恋爱/分手

#### 输出格式
```
💕 本周与【对方星座】的感情运势

今日信号：（1-2句分析对方心理）

建议：
1. 具体建议
2. 具体建议

避雷：（1件不要做的事）

时机：（最佳互动时间）

✨ 一句提醒：（暖心的话）
```

### 3.2 事业线运势

#### 输入字段
- 【我的星座】{my_sign} - 必选，12星座选择器
- 【对方星座/领导星座】{target_sign} - 必选，12星座选择器
- 【分析场景】{scene} - 必选，汇报/合作/冲突/日常

#### 输出格式
```
💼 本周与【对方星座】的事业运势

今日人设：（1句分析对方状态）

建议：
1. 具体建议
2. 具体建议

话术：（1-2句今天适合说的话）

避坑：（1件不要做的事）

时机：（最佳沟通时间）

✨ 一句策略：（犀利提醒）
```

## 5. 技术架构

### 前端
- 单个HTML页面（index.html）
- 两个Tab切换（感情/事业）
- 表单字段动态显示
- 加载状态提示
- 生成后展示结果
- 一键复制按钮

### 后端（Flask）
- 路由：GET / → 返回index.html
- 路由：POST /api/fortune → 接收参数，调用LLM，返回运势结果

### LLM调用
- 支持 DeepSeek / 豆包 API
- API配置通过 config.py 管理
- 实时生成，不存储数据
